# Rock-Paper-Scissor-JS
A simple Rock, Paper, Scissors game made for **learning purpose** with simple userInterface.

# Guide to play:
-Click on picture to throw what you want to throw either "rock", "paper" or "scissor".
-The computer will choose it's sign.
-Score will be displayed at the top of the button.
-The one who get's 5 points first will win the game.

Enjoy the game!
